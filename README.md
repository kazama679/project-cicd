# Todo List (React + Vite)

Ứng dụng Todo List đơn giản dùng React (Vite). Cho phép thêm, đánh dấu hoàn thành và xóa công việc. Dữ liệu lưu trong state (không có backend).

## Cấu trúc chính

- src/App.jsx: Trang chính
- src/components/TodoList.jsx: Danh sách công việc
- src/components/TodoItem.jsx: Từng mục công việc
- src/index.css: Styling đơn giản, responsive

## Chạy dự án

```bash
npm install
npm run dev
```

Sau đó mở URL được hiển thị (mặc định http://localhost:5173).

## Build

```bash
npm run build
```

## Tính năng

- Thêm công việc mới
- Đánh dấu hoàn thành
- Xóa công việc
- Responsive cho màn hình nhỏ
